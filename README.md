# sin-city
Library for ECDSA based construction for Anonymous Multi-Hop Locks (https://eprint.iacr.org/2018/472.pdf). Use `Cargo test` to try it. Currently tests exists for :
1) Setup and verify setup
2) Setup + Lock
3) Setup + Lock + Release
